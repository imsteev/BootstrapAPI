# bootstrap-flask-api


## Prerequisites

## Usage
```
python3 ./bootstrap_api.py -endpoints "endpoint1, endpoint2, endpoint3" -host localhost -port 8080
```

## Defaults
- host = "localhost"
- port = 8888

Note: all APIs will be generated with an `index` endpoint that can be reached at the root path aka `<host>:<port>`
